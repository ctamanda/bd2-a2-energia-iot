# Q1 - Painel operacional recente

## A consulta

```sql
SELECT a.equipe, r.status, r.prioridade, r.titulo, r.occurred_at
FROM energia.leituras r
JOIN energia.tecnicos a ON a.id = r.tecnicos_id
WHERE r.status IN ('aberto', 'em_andamento')
  AND r.occurred_at >= NOW() - INTERVAL '90 days'
ORDER BY r.prioridade DESC, r.occurred_at ASC
LIMIT 250;
```

## Diagnóstico

Esta consulta monta o painel operacional, listando os chamados em aberto ou em
andamento dos últimos 90 dias. O filtro recai sobre as colunas `status` e
`occurred_at` da tabela `leituras`, que tem 250 mil linhas e nenhum índice
nessas colunas.

Sem índice, o banco é obrigado a ler a tabela inteira (Seq Scan) e descartar a
maior parte das linhas pelo filtro. O gargalo é a leitura de disco: quase 10 mil
blocos são lidos só para encontrar algumas milhares de linhas relevantes.

## Hipótese

Como o filtro combina uma igualdade (`status`) com um intervalo de data
(`occurred_at`), acreditamos que um índice composto nessas duas colunas permita
ao banco localizar diretamente as linhas certas, sem precisar varrer a tabela
toda.

## Otimização aplicada

Criamos um índice composto em `(status, occurred_at)`:

```sql
CREATE INDEX idx_leituras_status_occurred
    ON energia.leituras (status, occurred_at);
ANALYZE energia.leituras;
```

## Resultado

- **Antes:** o banco usava Seq Scan, levava cerca de 318 ms e acessava quase
  10 mil blocos (sendo 8.699 lidos do disco).
- **Depois:** passou a usar Bitmap Index Scan, caindo para cerca de 110 ms e
  apenas 540 blocos.
- **Ganho:** aproximadamente 3 vezes mais rápido, com muito menos leitura de
  disco. Rodando a consulta repetidas vezes, com o cache já aquecido, o tempo
  cai para poucos milissegundos.

## Custo Colateral

- O índice ocupa cerca de 9,5 MB em disco.
- Operações de INSERT, UPDATE e DELETE na tabela `leituras` ficam ligeiramente
  mais custosas, pois o índice também precisa ser atualizado a cada escrita.
- Como o painel é consultado com muita frequência, o ganho em leitura compensa
  esse custo adicional de escrita.
